\begin{abstract}
Recovering clean anatomical structure from noisy magnetic resonance images is a practical image restoration problem with direct relevance to medical image analysis. This work evaluates a supervised denoising pipeline for paired T1 MRI volumes, where each noisy scan is matched with a clean target from the same subject. We train a 2D Residual U-Net on axial slices and use case-level data splits to prevent subject leakage. The full experiment includes 600 paired cases, divided into 420 training, 90 validation, and 90 test cases. On the held-out test set, the model reduces MAE from 0.0044 to 0.0028 and MSE from $3.72\times10^{-5}$ to $1.80\times10^{-5}$ relative to the noisy baseline. It also improves PSNR from 44.7757 dB to 47.7168 dB and SSIM from 0.9470 to 0.9930, with PSNR gains in all 90 test cases. These results show that residual slice-wise learning provides a simple, stable, and reproducible baseline for T1 MRI denoising.
\end{abstract}

\section{Introduction}

MRI denoising is a core image restoration problem. Given a noisy image $x$ and a clean target image $y$, the goal is to learn a mapping $f_\theta(x)$ that suppresses noise while preserving anatomical structure. This work focuses on paired T1 denoising because the input and target images belong to the same modality, have aligned spatial structure, and allow direct supervised training with standard restoration metrics.

We implement a 2D Residual U-Net baseline. U-Net architectures are widely used in biomedical image analysis because the encoder-decoder structure captures multi-scale context while skip connections preserve spatial detail \citep{Ronneberger2015UNet}. For denoising, residual prediction is natural: noisy and clean images share most anatomical content, so the network mainly needs to estimate a correction term rather than synthesize a full image from scratch. The experiment evaluates whether this baseline improves pixel-level error and structural similarity over the noisy input.

\section{Data}

The data root is \texttt{data/cn\_project\_t1\_noise2}. Each case directory contains:
\begin{itemize}
  \item \texttt{T1\_noisy.nii.gz}: the noisy T1 MRI volume;
  \item \texttt{T1\_clean.nii.gz}: the paired clean T1 MRI volume.
\end{itemize}

The full experiment includes 600 complete paired cases. We split the data at the case level, using a fixed random seed of 42. The resulting train, validation, and test sets contain 420, 90, and 90 cases, respectively. Case-level splitting is important because slice-level random splitting could place different slices from the same subject into both training and test sets, causing data leakage.

\section{Method}

\subsection{Preprocessing}

For each case, the noisy and clean NIfTI volumes are loaded as 3D arrays. Intensities are normalized per case by computing the 1st and 99th percentiles over the joint foreground voxels from both noisy and clean images, then clipping and scaling values to $[0,1]$. We extract axial 2D slices and skip near-empty slices. Each slice is resized to $256\times256$ before being passed to the model. Evaluation metrics are computed on the same normalized and resized 2D slices used by the model.

\subsection{Residual U-Net}

The model is a lightweight 2D U-Net with one-channel input and output. The encoder uses convolutional blocks and max pooling; the decoder uses transposed convolutions and skip connections. Instead of predicting the denoised image directly, the network predicts a residual $r_\theta(x)$:
\begin{equation}
  \hat{y} = \mathrm{clip}(x + r_\theta(x), 0, 1).
\end{equation}
The final residual convolution is initialized to zero, so the initial model behaves like the noisy baseline. This makes training stable because the network starts from an identity-like solution and learns only the denoising correction.

\subsection{Training objective}

The loss combines L1 and MSE terms:
\begin{equation}
  \mathcal{L}(\hat{y}, y) =
  \|\hat{y}-y\|_1 + 0.2\|\hat{y}-y\|_2^2.
\end{equation}
The L1 term encourages robust pixel-wise reconstruction, while the MSE term penalizes larger deviations. We use AdamW \citep{Kingma2015Adam} with learning rate $10^{-4}$ and weight decay $10^{-5}$. The model is trained for 40 epochs with batch size 24 on an NVIDIA RTX A6000 GPU. The checkpoint with the lowest validation loss is selected for final evaluation.

\section{Evaluation}

We compare the trained model against the noisy input baseline on the held-out test set. The metrics are mean absolute error (MAE), mean squared error (MSE), peak signal-to-noise ratio (PSNR), and structural similarity index (SSIM) \citep{Wang2004SSIM}. MAE and MSE are better when lower; PSNR and SSIM are better when higher. All reported values are case-level means over the 90 test cases.

\section{Discussion}

The results show that a 2D Residual U-Net is a strong baseline for this paired T1 denoising task. The model consistently improves pixel-wise error and structural similarity compared with the noisy input. The large SSIM gain suggests that the model does not merely smooth the image, but also preserves the main anatomical structures.

There are several limitations. First, the model processes axial slices independently and does not explicitly use 3D context from adjacent slices. A 2.5D model with neighboring slices as channels, or a 3D U-Net, may further improve consistency across slices. Second, the evaluation is performed on normalized resized 2D slices rather than original-resolution 3D volumes. Third, optional engineering improvements such as early stopping, mixed precision, and data augmentation are implemented in the codebase but are not part of the reported full-run results; using them in the final result would require retraining and updating the metrics.

\section{Conclusion}

This project implements a complete supervised T1 MRI denoising experiment. On 90 held-out test cases, the Residual U-Net improves MAE, MSE, PSNR, and SSIM over the noisy baseline, with PSNR improvement in all test cases. The experiment is simple, reproducible, and suitable as a course final-project baseline for medical image denoising.
